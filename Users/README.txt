папка проекта: Users\
описание: 
-скрипты генерируют запросы в формате JSON к заглушке на создание пользователя с параметрами логин и пароль,  получение пользователя по созданному id, получение всех пользователей.
-заглушка обрабатывает полученные запросы, формирует и отдает ответы в формате JSON

структура:
_data - файлы параметризации для LR

_scripts_jmeter - скрипты Jmeter
	генерирует клиентские запросы GET, POST на заглушку Java Spring Boot. 

_scripts_LR - скрипты LoadRunner Generator
	генерирует клиентские запросы GET, POST на заглушку Java Spring Boot. 

_scripts_java - скрипты на Java
	script_Users_RestAPI_v1 - заглушка на Java Spring Boot. Обрабатывает клиентские запросы по REST API (GET, POST). 


ATTENTION:
заглушка слушает порт 8090. 
скрипты работают на порт заглушки 8090.
